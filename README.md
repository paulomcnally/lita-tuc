# lita-tuc

Get balance from MPeso Tuc with Slack.

> Hoy le toca a {nombre}


## Installation
Add lita-tuc to your Gemfile:

``` ruby
gem 'lita-tuc'
```

## Test
``` bash
bundle exec rspec
```

## Usage

    @bot: tuc 00758888

## Build (development)

    gem build lita-tuc.gemspec

## Publish (development)

  gem push lita-tuc-x.x.x.gem
